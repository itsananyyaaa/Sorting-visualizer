## 🎬 Algorithm Visualizations

### Bubble Sort
```
Comparison: O(n²)
Best: O(n) - Already sorted
Worst: O(n²) - Reverse sorted
```
**Visualization**: Adjacent elements glow yellow, swap orange

### Selection Sort
```
Comparison: O(n²)
Best: O(n²) - Even if sorted
Worst: O(n²)
```
**Visualization**: Min element highlighted, swapped into position

### Insertion Sort
```
Comparison: O(n²)
Best: O(n) - Nearly sorted
Worst: O(n²) - Reverse sorted
```
**Visualization**: Elements shifted right, inserted left
