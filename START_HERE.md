🎯 5 SORTING ALGORITHMS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. BUBBLE SORT
     • Time: O(n²)
     • Simple and easy to follow
     • Good for learning basics
     • Shows every comparison

  2. SELECTION SORT
     • Time: O(n²)
     • Finds minimum repeatedly
     • Shows selection process
     • Fewer swaps than bubble

  3. INSERTION SORT
     • Time: O(n²)
     • Builds sorted portion gradually
     • Great on nearly-sorted data
     • Stable algorithm

  4. MERGE SORT
     • Time: O(n log n)
     • Divide and conquer
     • Consistent performance
     • Stable algorithm

  5. QUICK SORT
     • Time: O(n log n) average
     • Pivot-based partitioning
     • Generally fastest
     • Not stable

  └─ Current step / Total steps
