## OPTION 2: GITHUB PAGES (BEST - 5 MINUTES)

```
1. Create repo on GitHub
   github.com/new → Name: sorting-visualizer → Public

2. Push code:
   cd ~/Desktop/Sorting-visualiser/
   git init
   git add .
   git commit -m "Add sorting visualizer"
   git remote add origin https://github.com/YOUR_USER/sorting-visualizer.git
   git branch -M main
   git push -u origin main

3. Enable Pages:
   Repo → Settings → Pages → Branch: main → Save

4. Done! Site at:
   https://YOUR_USER.github.io/sorting-visualizer/ ✓
```

---

## OPTION 3: VERCEL (FAST - 2 MINUTES)

```
1. Run command:
   npm install -g vercel
   cd ~/Desktop/Sorting-visualiser/
   vercel

2. Follow prompts:
   - Project name: sorting-visualizer
   - Framework: Other
   - Root directory: ./
   - Build: (skip)

3. Done! Share link ✓

Result: https://sorting-visualizer.vercel.app/
```
